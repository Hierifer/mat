# Mat Terminal - 语音识别功能

## 📢 功能简介

Mat Terminal 支持实时语音转文字输入功能，让你可以通过语音快速输入命令和文本到终端。

## ✨ 特性

- ✅ **实时转录**：边说边转换成文字，实时显示
- ✅ **自动输入**：语音转录的文字自动发送到活动的终端窗格
- ✅ **快捷键控制**：`Ctrl+Shift+V` (Windows/Linux) 或 `Cmd+Shift+V` (macOS)
- ✅ **视觉反馈**：美观的浮动指示器显示录音状态
- ✅ **中文支持**：默认支持中文语音识别
- ✅ **浏览器原生**：使用 Web Speech API，无需安装额外组件

## 🎯 使用方法

### 快速开始

1. **激活语音输入**
   - 按下 `Ctrl+Shift+V` (Windows/Linux)
   - 或 `Cmd+Shift+V` (macOS)
   - 或点击终端窗口后按快捷键

2. **开始说话**
   - 看到浮动的语音指示器后，开始说话
   - 你的语音会实时转换成文字
   - 转换的文字会自动输入到当前活动的终端窗格

3. **停止录音**
   - 再次按 `Ctrl+Shift+V` / `Cmd+Shift+V`
   - 或点击指示器上的 ✕ 按钮

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+V` (Win/Linux)<br/>`Cmd+Shift+V` (Mac) | 开始/停止语音输入 |

## 🎨 界面说明

### 语音指示器

当语音识别激活时，右上角会显示一个浮动指示器：

```
┌─────────────────────────────┐
│ 🎤 正在聆听...           ✕ │
├─────────────────────────────┤
│ 转录的文字内容...            │
├─────────────────────────────┤
│ 按 Ctrl+Shift+V 停止        │
└─────────────────────────────┘
```

**状态指示**：
- 🔵 蓝色脉冲：正在录音
- 🔴 红色感叹号：发生错误
- ⚪ 灰色：已停止

## 🔧 技术说明

### Web Speech API

Mat Terminal 使用浏览器内置的 [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API)：

- **Chrome/Edge**: 完整支持，部分功能可离线使用
- **Safari**: 支持，需要在线
- **Firefox**: 支持有限

### 隐私保护

- ✅ 语音处理由浏览器完成
- ✅ 不会上传到 Mat Terminal 服务器
- ⚠️ 可能会发送到浏览器厂商的语音服务（如 Google）

## 🐛 故障排除

### 问题：语音识别不工作

**可能原因**：
1. 浏览器不支持 Web Speech API
2. 没有麦克风权限
3. 麦克风被其他应用占用

**解决方案**：
1. 使用 Chrome、Edge 或 Safari 浏览器
2. 检查浏览器麦克风权限设置
3. 关闭其他使用麦克风的应用

### 问题：识别不准确

**建议**：
1. 在安静的环境中使用
2. 使用质量好的麦克风
3. 说话清晰，语速适中
4. 避免方言或口音过重

### 问题：无法发送到终端

**检查**：
1. 确保有活动的终端窗格
2. 终端窗格是否被聚焦
3. 查看控制台是否有错误信息

## 🔒 安全性

### 权限要求

- **麦克风权限**：首次使用时浏览器会请求麦克风权限

### 数据隐私

- 语音数据仅在浏览器内处理
- 不会存储录音文件
- 转录历史会在停止录音后清除

## 📋 限制

1. **依赖浏览器**：需要支持 Web Speech API 的现代浏览器
2. **网络连接**：部分浏览器可能需要在线才能使用
3. **识别准确度**：取决于浏览器的语音识别引擎
4. **语言支持**：当前仅默认中文，其他语言需要配置
